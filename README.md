# Sorting Algorithms & Big O
## Holberton School Project

This repository contains implementations of various sorting algorithms and Big O notation analysis. It includes task-specific source files and documentation.

### Project Tasks

* **Task 0: Bubble Sort** -  Implementation of the Bubble Sort algorithm.
* **Task 1: Insertion Sort** - Implementation of the Insertion Sort algorithm for a doubly linked list of integers.
* **Task 2: Selection Sort** -  Impleentation of the Selection Sort algorithm for an array of integers.
* **Task 3: Quick Sort** -  Implementation of the Quic Sort algorithm for an array of integers.

---

### Technical Details

The project is implemented in the **C language**. All source code follows the **Betty** coding style and documentation standards.
